The Prysm:ID dashboard lives at [`app.prysmid.com`](https://app.prysmid.com). It's where you do everything admin without code: invite team members, register OAuth apps, connect social IdPs, brand the login, view audit log, manage billing.

Every page has its counterpart via API and via MCP — this section documents the UI path. If you prefer code, head to [/integrate](/en/integrate/your-saas-app/) or [/agents/tools](/en/agents/tools/).

## Layout

- **Left sidebar**: your workspaces. Click one to enter.
- **Tabs per workspace** (Overview, Apps, Users, IdPs, Branding, Billing, Audit, Settings).
- **Top right**: your avatar + theme switcher (light/dark).

## What's NOT in the dashboard (yet)

- **Create / edit end-user accounts** of the workspace. Users are created via OIDC signup in *your* app, not here. Admin user-management lives in API and MCP for now; UI lands 2026 Q3.
- **SAML SSO config UI**: today via API/MCP; dedicated UI on the roadmap.
- **Fine-grained usage metrics**: dashboard has basic counters; advanced analytics via audit log export.
